# PostosEletricos

A SwiftUI Map App, focused in Charger Stations for eletrical vehicles.

### Tecnologies applied:

- SwiftUI
- MVVM
- Clean Architecture
- MapKit
- Google Places
- Moya/Alamofire
- Lottie
